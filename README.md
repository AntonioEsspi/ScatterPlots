# ScatterPlots Using R Programming Language

In this lab, we were tasked to create a shiny app, using the shiny package
in R, using ui.R and server.R files. The app I created graphs a scatter plot,
were the user choses the x axis and y axis desired to be visualized and
helpful to better understand its correlation, the user chooses the x-axis
using a drop menu, and the y-axis is chosen using radio buttons. The user
also has the option to call for a linear regression line to be plotted to
whichever set of variables are currently plotted, this is to help the user
understand the data and the graph better.
The link: <https://antonio-esp1.shinyapps.io/Lab3/>
